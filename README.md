# Usability Testing Report (UI/UX)

## 📖 Project Overview

This project focuses on evaluating the usability of a mobile application through user testing and analysis. The goal is to identify usability issues, understand user behavior, and provide recommendations to improve the overall user experience.

---

## 🎯 Problem Statement

Many applications face usability challenges that can affect user satisfaction and task completion. This usability testing study aims to identify pain points and suggest design improvements based on user feedback and observations.

---

## 🎯 Objectives

- Identify usability issues
- Evaluate user satisfaction
- Analyze task completion efficiency
- Improve user experience

---

## 🛠 Tools Used

- Figma
- Usability Testing
- UX Research
- User Feedback Analysis
